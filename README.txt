Для локального разворачивания
1. Откройте терминал в корне вашего проекта и выполните следующую команду для сборки Docker-образа:
docker build -t my-static-site .
2. После успешной сборки образа вы можете запустить контейнер с помощью следующей команды:
docker run -d -p 8080:80 --name my-running-site my-static-site

-d: Запускает контейнер в фоновом режиме.
-p 8080:80: Перенаправляет порт 8080 на вашем хосте на порт 80 в контейнере.
--name my-running-site: Устанавливает имя для контейнера.

3. Откройте веб-браузер и перейдите по адресу:
http://localhost:8080
